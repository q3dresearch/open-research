# q3d research — Open Research

Public research outputs for the **Civic Flow Observatory (SG v0.1)** and future observatories.

## What's here

| Folder | Contents |
|---|---|
| `datasets/` | Dataset configs (`dataset.yaml`) — archetype, field mappings, source, tier |
| `signals/` | Signal configs (`signals.yaml`) — reusable signal templates per dataset |
| `runs/` | Run manifests (`run.yaml`) — provenance for every pipeline execution |
| `outputs/charts/` | Generated charts (PNG/SVG) |
| `outputs/drafts/` | Narrative drafts (markdown) |
| `outputs/eda/` | EDA reports per dataset |

## Data sources

All data is sourced from public open-data portals (e.g. [data.gov.sg](https://data.gov.sg)).
Each output credits the original source and links back.

## Methods

Pipeline methodology is defined via the `open-data-observatory-v1` and `research-director` skills,
referenced in `skills-lock.json`.
